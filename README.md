Password Strength Checker

A simple Python-based Password Strength Checker that evaluates the strength of a password based on common security rules such as length, uppercase and lowercase characters, numbers, and special characters.

This project demonstrates basic password security validation techniques used in cybersecurity and authentication systems.

Features

Checks password length

Detects uppercase letters

Detects lowercase letters

Detects numbers

Detects special characters

Provides feedback to improve password strength

Technologies Used

Python

Regular Expressions (Regex)

How It Works

The program analyzes the password entered by the user and checks whether it meets common security requirements. Based on the number of conditions satisfied, the password is classified as:

Weak

Moderate

Strong

Very Strong

It also provides suggestions to improve the password if it does not meet security standards.

How to Run

Clone the repository

git clone https://github.com/yourusername/password-strength-checker.git

Navigate to the folder

cd password-strength-checker

Run the program

python password_checker.py

Enter a password to check its strength.

Example
Enter a password: Hello123

Password Strength: Moderate

Suggestions:
- Add at least one special character
